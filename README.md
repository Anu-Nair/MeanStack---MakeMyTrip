# MEAN crud operation with Angular1 to make a website similar to makemytrip.com

In this web app we will be making a website similar to MakeMyTrip which performs CRUD operations for Flight bookings and Train bookings. 
Using this application you will be able to 
1. Add a flight/train
2. Display flight/train details
3. Delete a flight/train
4. Book tickets for the flight/train

# How to get this working
run npm install on the project folder/client to install dependencies.

run the project using npm start
